# 🤖 Crypto Bot Futures — BingX Trading Bot

## Что нового в v2.0

### 🎛 Полное управление через интерфейс
- Все настройки — API ключи, риски, стратегии, фильтры, whitelist/blacklist — прямо в GUI
- Не нужно лазить в конфиг!
- Тёмная тема, цветное логирование, таблицы позиций

### 🤖 Полная автономия
- Бот сам адаптируется под размер депозита (микро/малый/средний/крупный)
- Автоматическое определение плеча, риска, интервала сканирования
- Самовосстановление при ошибках API (backoff, retry)
- Circuit breaker при просадках

### 📊 Полное логирование
- Консоль + файл + JSON логи
- Отдельный лог решений (входы, выходы, пропуски)
- Лог всех API вызовов
- Лог состояния каждого компонента
- Логи в UI с фильтрами по уровню

### 📈 Адаптивность
- Адаптация под депозит: < $100, $100-1000, $1000-10000, > $10000
- Адаптация под рынок: смягчение фильтров при спокойном рынке
- Anti-martingale: снижение риска после убытков
- Weekend mode: снижение активности по выходным

## Установка

```bash
python setup.py
```

## Запуск

```bash
python main.py
```

## Настройка через GUI

1. Открой вкладку **⚙ Настройки**
2. Введи **API Key** и **API Secret** (BingX Futures)
3. Включи/выключи **Демо-режим**
4. Настрой риски, стратегию, фильтры
5. Нажми **💾 Сохранить настройки**
6. Нажми **▶ СТАРТ**

## Структура

```
├── main.py                 # Точка входа
├── setup.py                # Установка зависимостей
├── update_repo.sh          # Обновление на GitHub (Linux/Mac)
├── update_repo.bat         # Обновление на GitHub (Windows)
├── config/
│   └── bot_config.json     # Настройки (авто-генерируется)
├── logs/                   # Логи (авто-создаётся)
│   ├── tradingengine.log
│   ├── tradingengine.jsonl
│   └── decisions.jsonl
├── src/
│   ├── core/
│   │   ├── engine/         # Торговый движок
│   │   ├── executor/       # Исполнитель сделок
│   │   ├── exit/           # Менеджер выходов
│   │   ├── market/         # Данные и индикаторы
│   │   ├── risk/           # Риск-менеджмент
│   │   ├── scanner/        # Сканер рынка
│   │   └── trading/        # Позиции и ордера
│   ├── config/             # Константы и настройки
│   ├── intelligence/       # Стратегии
│   ├── ui/                 # PyQt5 интерфейс
│   └── utils/              # Утилиты
```

## Логи

Все логи сохраняются в папку `logs/`:
- `.log` — текстовые логи
- `.jsonl` — структурированные JSON логи для анализа
- `decisions.jsonl` — все решения бота
